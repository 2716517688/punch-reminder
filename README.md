# 打卡提醒 (Punch Reminder)

下班离开公司自动提醒打开纷享销客打卡。

## 功能

- 📍 手动标定公司 GPS 坐标
- 🔔 设定时间后（默认 19:00），离开公司超过阈值自动弹通知提醒
- 📱 检测到纷享销客打开后自动停止提醒
- ⚙️ 可调离开阈值（0-100m）和激活时间（17:00-22:00）
- 💾 设置自动保存，App 重启后恢复

## 使用

1. 在公司时打开 App，点击「标定公司坐标」
2. 调整离开阈值和激活时间
3. 点击「开始监听」
4. 授予所有权限（位置、后台位置、通知、电池优化、使用情况访问）

## 小米/MIUI 额外设置

- 设置 → 应用 → 打卡提醒 → 自启动：开启
- 设置 → 应用 → 打卡提醒 → 省电策略：无限制
- 设置 → 应用 → 打卡提醒 → 允许后台运行

## 编译

推送到 GitHub 后自动通过 GitHub Actions 编译，从 Actions → Artifacts 下载 APK。

## 技术栈

- Flutter 3.27
- geolocator（GPS 定位）
- flutter_local_notifications（本地通知）
- usage_stats（检测 App 使用情况）
- shared_preferences（本地存储）
